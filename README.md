# Login & Registration System (C++)

## 📌 Description
A simple Login and Registration System developed in C++. This project allows users to register with a username and password and then log in using the stored credentials. User data is stored in a text file using file handling.

## 🚀 Features
- User Registration
- User Login
- File Handling using `ifstream` and `ofstream`
- Duplicate Username Check
- Menu Driven Program
- Simple and Beginner Friendly

## 🛠️ Technologies Used
- C++
- File Handling
- Functions
- Loops
- Conditional Statements

## ▶️ How to Run
1. Compile the `main.cpp` file.
2. Run the program.
3. Choose:
   - Register
   - Login
   - Exit
